# Self-Construction OS · 自我构建 OS

A structured operating system for the transition from environmental conditioning toward conscious self-construction. Bilingual (EN / 中文), dark/light, blueprint-architectural aesthetic. Ten modules: a hero schematic of the conditioned human, the seven shaping forces + conditioning loop, awareness tools + 50-prompt reflection deck, dopamine-ladder + 21-day attention recovery, identity stack + 5-step redesign protocol, habit architecture grid, 9 cognitive distortions, window-of-tolerance + discomfort dosing, four-quadrant direction canvas, AI Self-Construction Guide, and a 36-month trajectory dashboard.

## Stack

Plain HTML, CSS, vanilla JavaScript. No build step. Bilingual via `[data-lang]` + CSS attribute selectors. Light/dark via `[data-theme]`. Blueprint-architectural aesthetic with Big Shoulders Display, DM Sans, Space Mono. All diagrams are inline SVG.
